Ex. 2: Writing a static method that takes positive integer n as input and outputs an array of n random integers in the range 0 -> 1,000,000

Output:



Source Code:

public class RandomInteger{
	public static void main(String[] args){
		int n=Integer.parseInt(args[0]);
		int[] a=new int[n];
		for (int i=0;i<n;i++){
			a[i]=(int)(Math.random()*1000000)+1;
			System.out.println("Position "+(i+1)+": "+a[i]);
		}	
	}
}
